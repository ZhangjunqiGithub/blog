е®ҳзҪ‘ең°еқҖпјҡNode.js (nodejs.org)

LSTдёәй•ҝжңҹзЁіе®ҡзүҲпјҢеҜ№дәҺиҝҪжұӮзЁіе®ҡжҖ§зҡ„дјҒдёҡзә§йЎ№зӣ®жқҘиҜҙпјҢжҺЁиҚҗе®үиЈ…LTSзүҲжң¬зҡ„Node.js

Currentдёәж–°зү№жҖ§е°қйІңзүҲпјҢеӯҳеңЁйҡҗи—Ҹзҡ„BugжҲ–е®үе…ЁжјҸжҙһгҖӮ

жҹҘзңӢе·Іе®үиЈ…зҡ„Node.jsзҡ„зүҲжң¬еҸ·

жү“ејҖз»Ҳз«ҜпјҢеңЁз»Ҳз«Ҝиҫ“е…Ҙе‘Ҫд»Ө node -v

Node.jsдёӯж №жҚ®жЁЎеқ—жқҘжәҗзҡ„дёҚеҗҢпјҢе°ҶжЁЎеқ—еҲҶдёәдәҶ3еӨ§зұ»пјҢеҲҶеҲ«жҳҜпјҡ

еҶ…зҪ®жЁЎеқ— з”ұNode.jsе®ҳж–№жҸҗдҫӣзҡ„пјҢдҫӢеҰӮпјҡfsгҖҒpathгҖҒhttpзӯү
иҮӘе®ҡд№үжЁЎеқ— з”ЁжҲ·еҲӣе»әзҡ„жҜҸдёӘ.jsж–Үд»¶пјҢйғҪжҳҜиҮӘе®ҡд№үжЁЎеқ—
з¬¬дёүж–№жЁЎеқ— з”ұз¬¬дёүж–№ејҖеҸ‘еҮәжқҘзҡ„жЁЎеқ—пјҢе№¶йқһе®ҳж–№жҸҗдҫӣзҡ„еҶ…зҪ®жЁЎеқ—пјҢд№ҹдёҚжҳҜз”ЁжҲ·еҲӣе»әзҡ„иҮӘе®ҡд№үжЁЎеқ—пјҢдҪҝз”ЁеүҚйңҖиҰҒдёӢиҪҪ
